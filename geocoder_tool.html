🌍 Hotel Geocoder Tool

Converts hotel addresses to exact GPS coordinates using OpenStreetMap Nominatim API

📋 How to use

  1. Upload your hotels_datasheet.csv file below
  2. Click Start Geocoding and wait (~18 min for 1069 hotels)
  3. Download the updated CSV when done — it replaces the city-level coords with exact hotel GPS pins
  4. Upload the new CSV to your Byethost project
⚠️ Important: Keep this tab open and active while running. Nominatim allows 1 request/second — do not close or refresh the page during geocoding.
